# Basic React

Basic React는 React 학습 과정을 담은 레포지터리입니다. 각각의 폴더는 독립적인 프로젝트로 구성되어 있으며, React의 주요 개념을 학습하고 실습하는 과정을 담고 있습니다.

### 💡 목표

    1. React의 주요 개념 이해 및 실습
    2. 컴포넌트 기반 설계와 상태 관리 방법 학습
    3. 성능 최적화 및 프로젝트 배포 경험 쌓기

<br>
